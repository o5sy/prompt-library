## 사용 방법
1. `Ctrl + F` (또는 GitHub의 상단 검색창)를 눌러 키워드를 검색합니다.
2. 원하는 파일(.md)로 이동합니다.
3. 코드 블록의 **[Copy]** 버튼을 눌러 복사 후 사용합니다.

## 프롬프트 추가 규칙
1. 새로운 프롬프트는 `template.md` 양식을 복사하여 생성합니다.
2. 파일명은 `영문-케밥-케이스.md` (예: `daily-fortune-teller.md`) 형식을 권장합니다.
3. 실행해 보고 결과가 좋았던 설정(온도, 모델 등)은 'Usage Tip'에 가볍게 메모합니다.
