# 서양 점성술 네이탈 차트 분석

> **용도:** Astro-seek의 개인 천궁도(Natal Chart) 데이터를 분석하여 심층 운세 및 인생 변곡점 확인
> **출처:** [인스타그램 @dopine_lessgo](https://www.instagram.com/p/DRGrFXgj4v_/?utm_source=ig_web_copy_link&igsh=MzRlODBiNWFlZA%3D%3D)

## 📋 Prompt

### 1단계: 데이터 입력

Astro-seek 사이트에서 복사한 개인 차트 데이터(`Planet positions` ~ `Other aspects`)를 먼저 붙여넣은 후, 아래 질문 중 하나를 입력하세요.

### 2단계: 질문 프롬프트 예시

```text
이 차트를 분석해서 인생의 변곡점을 알려주고 10년 단위 운세 흐름도 보여줘.
```

```text
내 직업적 재능과 잘 맞는 분야를 알려줘.
```

```text
시기별 조심해야 할 점은 무엇인가요?
```

```text
연도별 금전운 흐름을 분석해줘.
```

```text
2026년을 월별로 나눠서 흐름을 예측해줘.
```

## 💡 Usage Tip

* **데이터 추출 최적화**: Astro-seek 사이트의 'Extended settings'에서 House system을 반드시 **'Whole Sign'**으로 변경한 뒤 'AI/ChatGPT Data Export' 버튼을 눌러야 가장 체계적인 데이터가 추출됩니다.
* **정확도 향상**: 기존 대화의 편향을 피하고 싶다면 로그아웃 상태나 시크릿 모드에서 실행하는 것이 더 높은 정확도를 보여주는 경향이 있습니다.
* **확장 질문**: 기본 운세 외에도 이사운, 취직, 시험 등 구체적인 고민을 덧붙이면 AI를 브레인스토밍 파트너로 더 깊게 활용할 수 있습니다.
